# Equazioni di secondo grado
## $21x^2-10x+1=0$

> [!NOTE] Soluzione
> $\Delta = 100-84 = 16$
> $x_1 = \frac{10+4}{42} = \frac{14}{42} = \frac{1}{3}$
> $x_2 = \frac{10-4}{42} = \frac{6}{42} = \frac{1}{7}$

## $x^2-6x-16=0$

> [!NOTE] Soluzione
> $\Delta = 36+64 = 100$
> $x_1 = \frac{6-10}{2} = -\frac42 = -2$
> $x_2 = \frac{6+10}{2} = \frac{16}{2} = 8$
